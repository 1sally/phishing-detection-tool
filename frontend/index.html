The scanner uses the trained phishing model from the Flask backend.

F.A.Q

  • What is phishing?

    Phishing is a cybercrime in which an attacker impersonates a trusted brand or person to trick users into sharing passwords, payment data, or other sensitive information.

  • How does this scanner work?

    The app extracts URL and page-based features from the submitted link, then sends them to the trained machine learning model to estimate whether the site is safe or suspicious.

  • What should I do if the result is risky?

    Avoid logging in, entering card details, or downloading files until you verify the URL independently. Check the domain carefully and compare it with the official website.

About PhishDetector

This project helps users validate website URLs with a machine learning model that flags suspicious phishing behavior before they decide to continue.
